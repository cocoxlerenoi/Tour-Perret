Petit guide de fonctionnement

Cette application permet de créer un scénario (jeu de lumières) à diffuser sur la Tour Perret.

  • Les effets : Les effets sont des blocs à déplacer sur la ligne de temps.

  • Editer un effet : Une fois l'effet sur la ligne de temps vous pouvez cliquer dessus pour faire apparaître une fenêtre d'édition.

  • Enregistrer un scénario : Vous pouvez enregistrer jusqu'à 5 scénarios différents sur le périphérique que vous utilisez.

  • Envoyer un scénario : Vous pouvez envoyer un scénario à jouer sur la tour une fois qu'il est terminé. Attention vous ne pourrez plus le modifier une fois envoyé !
